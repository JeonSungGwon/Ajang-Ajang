# 아장아장(아이들의 장터)
## 프로젝트 소개
- 안전하고 편리한 육아 중고장터 플랫폼으로 중고 육아용품을 편리하게 거래할 수 있고, 다양한 기능을 통해 사용자의 프라이버시 및 안전성을 지켜주는 프로젝트 입니다.

## 사용 도구

- 이슈관리 : Jira
- 형상관리 : GitLab
- 디자인 : Figma
- CI/CD : Jenkins

## 개발 도구

- Visual Studio Code : 1.91.0
- IntelliJ : 2024.1.4 (Ultimate Edition)
